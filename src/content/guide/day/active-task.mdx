## Description

This is the active task layout. Tasks in this state are incomplete and allow full editing, reordering, and manipulation (e.g. moving or duplicating to another day).

## Buttons

1. Drag to reorder the task (icon appears only on hover).
2. Delete the task.
3. Duplicate the task to the next day.
4. Move the task to the next day.
5. Mark the task as complete.

## Inputs

1. I. Task title input field.
2. II. Task note input field.
