## Description

Completed tasks are grouped separately from active ones. While marked complete, they can still be reordered, duplicated, or reactivated as needed.

## Buttons

1. Drag to reorder the task (icon appears only on hover).
2. Delete the task.
3. Duplicate the task to the next day.
4. Mark the task as active.

## Labels

1. A. Task title (read-editable).
2. B. Task note (read-editable).
