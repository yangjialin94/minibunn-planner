## Description

The Password panel allows users to securely update their account password. Navigation between other account panels is also available from this view.

## Buttons

1. Switch to the **Subscription** panel.
2. Switch to the **Password** panel.
3. Switch to the **Support** panel.
4. Change the user&apos;s password.
