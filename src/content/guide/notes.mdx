## Description

The Notes panel allows users to jot down general-purpose notes that are not tied to specific dates. Notes can be reordered, edited, or removed anytime.

## Buttons

1. Add a new note.
2. Delete the note.
3. Drag to reorder the note (icon appears only on hover).

## Inputs

1. I. Note content input field.

## Labels

1. A. Total notes count.
2. B. Note creation date.
